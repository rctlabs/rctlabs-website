# Introducing JITNA: A Language for Intent

Today, we're thrilled to announce the public release of JITNA (Just In Time Natural Analysis), our domain-specific language for specifying and analyzing intent in AI systems.

## What is JITNA?

JITNA is a language designed to make it easier to explicitly specify intent in AI systems. Instead of relying on implicit goals embedded in training data or reward functions, JITNA allows developers to write out their intent clearly and formally.

## Why We Built It

As we worked with teams across different industries, we noticed a consistent challenge: how to clearly communicate the intent that AI systems should optimize for. JITNA addresses this by providing a concrete syntax and semantics for intent specification.

## Key Features

- **Clear Syntax** - Easy for humans to read and write
- **Formal Semantics** - Unambiguous interpretation by machines
- **Composable** - Build complex intents from simpler building blocks
- **Verifiable** - Analyze and verify that implementations match specifications

## Getting Started

You can find JITNA documentation, examples, and tools at our GitHub repository. We've included sample projects to help you get started.

## Open Source

JITNA is released under an open source license. We welcome contributions, feedback, and improvements from the community.
